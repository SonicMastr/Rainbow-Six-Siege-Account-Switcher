Welcome to Siege Account Switcher

This program allows users of both the Steam and Uplay Versions of Rainbow Six Siege to seamlessly switch between both accounts without having to install the game twice. This gives players the ability to save 70-100GB of storage that is occupied by a second installation.




To start installation to make account switching a two-click process, click next.


If you just want to switch between Uplay accounts,
click the Uplay Only button

Delete the Uplay Version of Siege

While logged in with your Uplay only account

In Uplay, Navigate to

Rainbow Six Siege > Properties > Local Files

and click "Uninstall"

Select Your Steam Siege Install Folder

Please click the "Select Game Folder" button and navigate to your Siege Steam directory

For me, it was

C:/Program Files (x86)/Steam/steamapps/common/ Tom Clancy's Rainbow Six Siege/



Locate Installed Game

Back in Uplay, select the "Locate installed game" option underneath the Download button. When asked for you folder, remember your Steam install folder

You may get a prompt to verify files. Confirm the verification and wait for it to finish. Once that is done, click next.


(You can check your selected path by clicking the back button. Everything up till now has been saved)

You're all Setup!

Now you can use this application to the fullest extent. Click "Finish" to complete the setup.
